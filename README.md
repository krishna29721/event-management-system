# Event Management System

A web-based system to manage events, registrations, and ticketing.

## Features
- Event creation & management
- User registration & login
- Event booking system
- Ticket pricing
- Capacity control
- Payment simulation

## Technologies Used
- PHP
- MySQL
- HTML, CSS, Bootstrap

## How to Run
1. Install XAMPP
2. Place folder in htdocs
3. Create database event_db
4. Run project on localhost
